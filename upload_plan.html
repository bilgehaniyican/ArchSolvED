ArchSolvED

Upload Site Plan




- At this stage, the designer is asked to upload a vector drawing of the site plan where the educational building will be designed.

- The file format has to be ".dxf" yet the file name may vary

- Drawing units must be in meters.

- The boundaries should be defined by an enclosed area with the "POLYGON" command.

- The software needs definitions of the site boundaries in the drawing file (.dxf) in specific layer names.

- "siteboundary" and the "setbackboundary" layer names must be used.
